# BeastLocator-flutter: 淫夢のみちびき

これを読んでいるならば、野獣邸、という言葉を絶対に一度は聞いたことがあると思います  
このアプリを使えば、その聖地への方角をすぐに知ることができるゾ<br><br>

**⚠️このアプリは、個人的に制作したファンメイドアプリです。コートコーポレーション本社周辺での迷惑行為を推奨、および助長するものではありません。**<br>

## 概要

>**「こ↑こ↓」**  
>**「はえ～、すっごい大きい…」**<br>
<br>
野獣邸の方角を知れる、ただそれだけのアプリです<br>
野獣邸は基本的に、聖地巡礼することはタブーとされており、実際野獣の日に来るなという公式の声明も出ています<br>
そんな野獣邸を身近に感じたかったので、このアプリを作りました<br>
<br>

Flutterなので、iOS/Android/macOS/Windows/OpenHarmonyなどの各プラットフォームへ容易に対応できます。

iOS版に関しては、TestFlightから簡単にインストールすることが可能です。
[TestFlight (0.9.5)](https://testflight.apple.com/join/6heARakB)

## 主な機能

  なんだこれは... たまげたなあ

 - 野獣邸への方角・距離表示
 - 野獣邸までの直線距離を表示するウィジェット
 - 野獣邸到達時の通知
 - 野獣邸から一定の距離に達した際、強制的に音声を再生　~~やめてくれよ...~~
 - iOSでのLive Activityおよび対応端末でのDynamic Islandによる方角・距離表示
 - ホーム画面とロック画面のウィジェット機能（不完全）
 - 使える機能はスマホ版と同等、Apple Watch版のBeastLocator

## 実装予定の機能
 - 聖地巡礼機能（野獣邸を除く）
 - 淫夢にまつわる-ご当地MAD機能
 - 自由に聖地を追加し、ユーザーと共有できる機能
 - INP Map（淫夢ナビ）

## その他の機能

さすがに野獣邸に実際に行って動作確認するのはまずいので、デバッグメニューから目的地の座標を変更できるようにしています
